# Security Policy

## Supported versions

This project is early-stage research software. Only the latest commit on the default branch is supported.

## Reporting a vulnerability

If you believe you have found a security vulnerability:

1. **Please do not** open a public issue with exploit details.
2. Prefer creating a **GitHub Security Advisory** for the repository (if enabled).
3. If advisories are not available, open a GitHub issue with *minimal* information and request a private contact channel.

We will do our best to acknowledge reports and ship a fix.

## Safety notice

This repository is **not** a certified medical device and is provided "as-is".
Use in clinical settings is at your own risk and subject to your local laws and regulations.
